Hello! My name is Andrew and this is my React JS Carousel component!

The Carousel:
  1.Works for mobile and desktop devices
  2.Supports swipes(only for mobile version)
  3.Works for any HTML content
  4.Is animated
  5.Supports infinite option
  6.Supports scrolling to a selected slide(only for desktop version)

This component is made from scratch, without using any libraries.

To setup this component you should:
1.Get the core files from https://github.com/Eeegry/carousel
2.Install Webpack
3.Install Webpack Cli
4.Install needed packages, you can find them in package.json file. It is recommended to use "npm install" command.
5.All needed files, modules and packages are installed! You can launch the app using npm start co—åmand.
